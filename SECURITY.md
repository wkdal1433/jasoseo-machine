## <img src="https://raw.githubusercontent.com/lucide-icons/lucide/main/icons/alert-circle.svg" width="24" height="24" valign="bottom" /> Reporting a Vulnerability

본 프로젝트에서 보안 취약점(예: API 키 노출 위험, 로컬 파일 접근 취약점 등)을 발견하셨다면, 공개적인 Issue 탭 대신 아래 메일 주소로 직접 보고해 주시기 바랍니다.

- **Email:** [wkdal1433@naver.com](mailto:wkdal1433@naver.com) (임시 메일주소)

보고 시 아래 내용을 포함해 주시면 빠른 대응에 큰 도움이 됩니다:
- 취약점의 유형 및 영향 범위
- 취약점을 재현할 수 있는 상세 단계 또는 PoC 코드
- 해결을 위한 권장 사항 (선택 사항)

보고해 주신 내용은 검토 후 최대한 빠른 시일 내에 수정 패치를 배포하겠습니다. 1인 개발 프로젝트의 특성상 대응이 다소 늦어질 수 있는 점 양해 부탁드립니다.
