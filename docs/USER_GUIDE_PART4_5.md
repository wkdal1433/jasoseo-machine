# <img src="https://unpkg.com/lucide-static@latest/icons/wand-sparkles.svg" width="32" height="32" valign="bottom" /> 자소서 머신 사용 설명서 (Part 4 & 5: 실전 작성 및 자동 제출)

축하합니다! 이제 당신의 에피소드들이 준비되었습니다. 이제 자소서 머신의 핵심인 **9-Step 생성 파이프라인**을 통해 실제 자기소개서를 완성하고, 채용 사이트에 클릭 한 번으로 제출해 보겠습니다.

---

## <img src="https://unpkg.com/lucide-static@latest/icons/wand-sparkles.svg" width="24" height="24" valign="bottom" /> Part 4. 자소서 작성 마법사 (Wizard 실전)

### 4-1. 새 지원서 작성 시작
1.  대시보드 상단의 **[새 지원서 작성]** 버튼을 클릭합니다.
2.  회사명, 지원 직무, 채용 공고(복사 후 붙여넣기), 그리고 자소서 문항(글자 수 제한 포함)을 입력합니다.

> **[캡처 포인트 8: 지원서 기초 설정 화면]**
> *   화면: Application Setup 페이지
> *   내용: 삼성전자, 현대자동차 등 가상의 회사 정보와 공고가 입력된 모습
> *   강조: 우측 하단의 **[지원서 분석 시작 ➔]** 버튼

### 4-2. 9단계 생성 파이프라인 (Step 0 ~ 8)
분석 시작을 누르면 AI가 각 단계별로 작업을 수행합니다.

*   **Step 0 (기업 분석):** 공고를 분석해 핵심 HR 의도(실행력, 안정성 등)를 도출합니다.
*   **Step 1 (질문 재해석):** 질문의 의도를 파악하여 최적의 답변 방향을 설정합니다.
*   **Step 2 (에피소드 매칭 - 중요!):** AI가 내 라이브러리에서 가장 적합한 에피소드를 추천합니다. 내용을 확인하고 **[승인(Approve)]** 버튼을 누르세요.
*   **Step 3~5 (생성 스트리밍):** AI가 실시간으로 자소서를 써 내려갑니다. 화면에 글자가 촤르륵 채워지는 모습을 확인하세요!
*   **Step 6~8 (검증):** 탈락 패턴(오타, 부적절한 표현)과 이중 코딩(내 역량+회사 가치) 여부를 최종 검증합니다.

> **[캡처 포인트 9: 실시간 생성 스트리밍 화면]**
> *   화면: Step 3~5 작업창
> *   내용: AI가 실시간으로 문장을 생성하고 있는 역동적인 모습
> *   강조: 우측의 **에피소드 원문 대조** 뷰와 생성 중인 텍스트

## <img src="https://unpkg.com/lucide-static@latest/icons/rocket.svg" width="24" height="24" valign="bottom" /> Part 5. 사이트 연동 및 자동 제출

### 5-1. 전체 검토 및 최종 수정 (Full Review)
모든 문항 작성이 완료되면 전체 자소서를 한눈에 볼 수 있는 화면으로 이동합니다.

1.  각 문항의 내용을 최종 확인합니다.
2.  수정이 필요한 부분은 즉시 편집하여 저장합니다.

> **[캡처 포인트 10: 전체 검토(Full Review) 화면]**
> *   화면: Full Review 페이지
> *   내용: 1번부터 N번 문항까지 깔끔하게 정리된 자소서 전문
> *   강조: 상단 우측의 **[채용 사이트로 전송]** 버튼

### 5-2. 크롬 확장 프로그램으로 자동 입력하기
이제 마지막 마법입니다. 앱에서 쓴 글을 채용 사이트 폼에 자동으로 채워 넣습니다.

1.  Chrome 브라우저를 열고, **실제 채용 사이트의 자소서 입력 페이지**로 이동합니다.
2.  자소서 머신 앱의 우측 상단 **[채용 사이트로 전송]** 버튼을 클릭합니다.
3.  크롬 브라우저에서 **자소서 머신 브릿지** 확장이 폼을 감지하고, 해당 문항의 내용을 자동으로 입력합니다!

> **[캡처 포인트 11: 실제 채용 사이트 자동 입력 화면]**
> *   화면: 실제 웹 브라우저(예: 사람인 폼)와 자소서 머신 앱이 나란히 있는 화면
> *   내용: 빈 입력창에 자소서 내용이 자동으로 촤르륵 채워져 있는 모습
> *   강조: 크롬 상단 툴바의 **자소서 머신 확장 아이콘**이 활성화(초록색)된 상태
