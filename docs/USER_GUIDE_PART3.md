# 자소서 머신 사용 설명서 (Part 3: 나의 무기 만들기)

환경 설정이 끝났다면 이제 AI가 나를 대신해 글을 쓸 수 있도록, 나의 경험 데이터를 입력할 차례입니다. 자소서 머신은 단순한 텍스트 입력이 아닌, **지능형 온보딩**과 **AI 인터뷰**를 통해 당신의 무기를 날카롭게 다듬어 줍니다.

---

## 📦 3-1. 매직 온보딩 (Magic Onboarding)
기존에 가지고 있던 이력서나 자기소개서 파일을 업로드하면, AI가 이를 분석하여 핵심 프로필과 에피소드를 자동으로 추출합니다.

1.  대시보드 또는 프로필 메뉴에서 **[이력서 업로드]** 버튼을 클릭합니다.
2.  PDF, Word, 또는 MD 파일을 선택하여 업로드합니다.
3.  AI가 파일을 분석하는 동안 잠시 기다리면, 이름, 학력, 기술 스택, 그리고 주요 프로젝트 경험이 자동으로 정리됩니다.

> **[캡처 포인트 5: 이력서 업로드 및 추출 화면]**
> *   화면: 파일 탐색기에서 이력서 파일을 선택하는 모습, 혹은 업로드 후 프로필 카드가 생성된 대시보드 화면
> *   내용: "이력서 분석 완료! 5개의 에피소드가 추출되었습니다." 같은 알림 메시지
> *   강조: 생성된 **프로필 카드**와 **추출된 기술 스택** 영역

---

## 🎙️ 3-2. 에피소드 마이닝 (Episode Interviewer)
추출된 내용이 부족하거나 새로운 경험을 추가하고 싶을 때, AI와 대화하며 경험을 구조화할 수 있습니다.

1.  에피소드 메뉴에서 **[새 에피소드 발굴하기]**를 클릭합니다.
2.  AI가 질문을 던집니다: "최근에 기술적으로 가장 어려웠던 경험은 무엇인가요?"
3.  친구와 대화하듯 편하게 답변을 입력하세요. AI가 꼬리 질문을 통해 **S-P-A-A-R-L(상황-문제-행동-판단-결과-교훈)** 구조로 내용을 정리해 줍니다.
4.  인터뷰가 끝나면 자동으로 구조화된 에피소드 카드가 라이브러리에 저장됩니다.

> **[캡처 포인트 6: AI 인터뷰 화면]**
> *   화면: 에피소드 발굴 마법사(Discovery Wizard) 채팅창
> *   내용: AI의 질문과 유저의 답변이 오가는 채팅 내역
> *   강조: 우측 하단에 실시간으로 빌드업되고 있는 **에피소드 요약 미리보기**

---

## 🗃️ 3-3. 에피소드 라이브러리 관리
저장된 에피소드들은 나중에 자소서를 쓸 때 AI가 골라 쓸 수 있는 **'원천 데이터'**가 됩니다.

*   **에피소드 카드:** 각 카드에는 제목, 직무 역량, HR 의도(실행력, 성장성 등)가 태그로 붙어 있습니다.
*   **편집 및 삭제:** 언제든지 카드를 클릭하여 내용을 수정하거나, 더 이상 필요 없는 경험은 삭제할 수 있습니다.

> **[캡처 포인트 7: 에피소드 라이브러리 목록]**
> *   화면: Episode List 페이지
> *   내용: 여러 개의 에피소드 카드가 그리드 형태로 나열된 모습
> *   강조: 각 카드에 붙어 있는 **HR 의도 태그(Execution, Growth 등)**
