# 자소서 머신 사용 설명서 (Part 1 & 2: 설치 및 환경 설정)

본 가이드는 AI를 활용해 서류 합격률을 극대화하는 **자소서 머신**의 초기 설정 방법을 다룹니다. 가이드를 따라 차근차근 설정을 완료해 보세요!

---

## 🚀 Part 1. 시작하기 전에 (필수 환경 설정)

자소서 머신은 똑똑한 두뇌(AI)를 빌려와 작동합니다. 이를 위해 내 컴퓨터에 AI 엔진을 설치하는 과정이 필요합니다.

### 1-1. Node.js 설치 (기본 엔진)
자소서 머신의 구성 요소를 구동하기 위한 기본 환경입니다.
*   **다운로드:** [Node.js 공식 홈페이지](https://nodejs.org/)에서 **LTS 버전**을 설치해 주세요.

### 1-2. AI 엔진(CLI) 설치
자소서 머신이 지원하는 두 가지 AI 엔진 중 하나 이상을 설치해야 합니다. (둘 다 설치하는 것을 권장합니다.)

#### 🔹 Gemini CLI (구글)
1.  **API 키 발급:** [Google AI Studio](https://aistudio.google.com/app/apikey)에서 무료 API 키를 발급받으세요.
2.  **설치:** 터미널(CMD 또는 PowerShell)을 열고 아래 명령어를 입력합니다.
    ```bash
    npm install -g @google/gemini-cli
    ```
3.  **인증:** `gemini auth`를 입력하고 발급받은 API 키를 붙여넣으세요.

#### 🔸 Claude Code (앤스로픽)
1.  **설치:** 터미널에 아래 명령어를 입력합니다.
    ```bash
    npm install -g @anthropic-ai/claude-code
    ```
2.  **인증:** `claude`를 입력하여 로그인을 진행하세요.

### 1-3. 설치 확인하기 (중요!)
제대로 설치되었는지 확인하기 위해 터미널에서 다음 명령어를 입력해 보세요.

> **[캡처 포인트 1: 터미널 설치 확인 화면]**
> *   화면: 검은색 터미널 창
> *   내용: `gemini --version`과 `claude --version`을 입력했을 때 버전 숫자가 출력되는 모습
> *   강조: 출력된 버전 숫자 부분에 빨간색 박스

## 📦 Part 2. 프로그램 설치 및 실행

### 2-1. 자소서 머신 앱 실행
1.  다운로드 받은 **자소서 머신 패키지**의 압축을 해제합니다.
2.  폴더 내의 `jasoseo-machine.exe`(또는 실행 파일)을 더블 클릭하여 실행합니다.

> **[캡처 포인트 2: 앱 첫 실행 화면]**
> *   화면: 자소서 머신 대시보드 초기 화면
> *   내용: "설정되지 않음" 경고나 비어 있는 대시보드 모습
> *   강조: 우측 상단의 **설정(톱니바퀴 아이콘)** 버튼

### 2-2. 크롬 확장 프로그램 등록 (웹사이트 연동용)
앱에서 쓴 자소서를 채용 사이트에 자동으로 입력하려면 크롬 확장이 필요합니다.

1.  Chrome 브라우저 주소창에 `chrome://extensions`를 입력합니다.
2.  우측 상단의 **개발자 모드** 스위치를 켭니다.
3.  좌측 상단의 **[압축해제된 확장 프로그램을 로드합니다]** 버튼을 클릭합니다.
4.  자소서 머신 패키지 폴더 안에 있는 `extension` 폴더를 선택합니다.

> **[캡처 포인트 3: 크롬 확장 프로그램 설정 화면]**
> *   화면: 크롬 확장 프로그램 관리 페이지
> *   내용: 개발자 모드가 켜져 있고, **'자소서 머신 브릿지'** 카드가 등록된 모습
> *   강조: '개발자 모드' 스위치와 '로드' 버튼 위치

---

## ⚙️ Part 2-3. 앱과 AI 엔진 연결하기

앱을 실행했다면, 설치한 Gemini/Claude가 앱과 잘 연결되는지 확인해야 합니다.

1.  앱의 우측 상단 **설정(톱니바퀴)** 메뉴로 들어갑니다.
2.  **AI Provider 설정** 섹션에서 내가 설치한 엔진의 경로(보통 `gemini` 또는 `claude`)가 입력되어 있는지 확인합니다.
3.  각 엔진 옆의 **[Test Connection]** 버튼을 누릅니다.

> **[캡처 포인트 4: 앱 설정 화면]**
> *   화면: Settings 페이지
> *   내용: Gemini/Claude 경로 입력창과 Test 버튼
> *   강조: 테스트 성공 후 나타나는 **'Connection Successful'** 초록색 메시지
