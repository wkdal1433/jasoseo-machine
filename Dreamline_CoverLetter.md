# 드림라인(주) 자기소개서 (자유 양식)

**지원 분야:** ICT 기술 기획 (신입)
**컨셉:** AI 시대의 생산성을 증명하는 기획자 (Vibe Coding Pioneer)

---

## 1. 지원동기: 보이지 않는 곳에서 세상을 움직이는 '디지털 동맥'을 설계하고 싶습니다.
**[Why Dreamline? 인프라의 가치를 아는 엔지니어의 선택]**
화려한 앱 서비스 뒤에는 언제나 끊기지 않는 데이터 고속도로가 존재합니다. 저는 **MediaPolytech** 프로젝트에서 로컬 저장소를 AWS S3로 마이그레이션하며, 안정적인 인프라가 서비스의 확장성을 결정짓는 핵심 요소임을 뼈저리게 경험했습니다. 이 거대한 '디지털 동맥'을 깔고 운영하는 드림라인의 기술력에 깊은 경외심을 느꼈으며, 이제는 단순한 사용자를 넘어 이 인프라를 가장 효율적으로 설계하고 비즈니스 가치로 연결하는 기획자가 되고자 지원했습니다.

**[Why Planner? 기술의 언어를 비즈니스의 가치로 통역하다]**
개발자로 시작했지만, 저의 열정은 코드를 넘어 '서비스 전체'를 향했습니다. **UMC 활동 당시 안드로이드 파트장**을 맡아 기획자, 디자이너와 협업하며 "기술적으로 안 돼요"라는 말 대신 대안을 제시하는 소통의 중요성을 배웠습니다. "기술적으로 불가능한 기획"과 "비즈니스 맥락을 모르는 개발" 사이의 간극을 보며, **"기술의 가능성과 한계를 명확히 알고 조율하는 기획자"**의 필요성을 절감했습니다. 저는 이 간극을 메우고, 드림라인의 기술력이 온전히 비즈니스 성과로 이어지도록 돕는 '기술 통역사'가 되겠습니다.

## 2. 직무 핵심 역량: 기술을 '서비스'로 만드는 실행력
**[Vibe Coding: 1인 애자일로 증명한 '초속 생산성']**
저는 AI Agent(Claude Code)와 바이브 코딩(Vibe Coding)이라는 개발 패러다임의 변화를 빠르게 포착했습니다. **"개발 지식을 갖춘 기획자가 AI와 협업할 때 생산성의 한계는 없다"**는 가설을 검증하기 위해 **DayScript** 프로젝트를 주도했습니다. 기획 직후 AI와 협업하여 TypeScript 기반 로직을 생성하고, 실시간 피드백으로 설계를 수정하는 **초단기 애자일 프로세스**를 확립했습니다. Figma 설계부터 구현까지의 전 과정을 압도적인 속도로 완주하며, 기술을 아는 기획자의 비즈니스 실현 속도를 증명했습니다. (※ 별첨: 기획/설계 Figma 링크)

**[Trend Insight: API를 넘어 '인프라'를 구축해본 경험]**
남들보다 한발 앞선 기술 도입으로 차별화된 결과를 만들어냅니다. 대학교 인공지능 수업 당시, 대다수가 생성형 AI의 단순 API 활용에 그칠 때 저는 최신 기술인 **3D Gaussian Splatting**에 주목했습니다. 단순히 이론에 그치지 않고 학우들이 이 기술을 직접 체감할 수 있도록, **학교 실습실 컴퓨터에 직접 오픈소스 모델을 구축하여 렌더링 서버를 띄우고 프론트엔드 뷰어까지 개발**했습니다. 남들이 가지 않은 길을 선택해 **'3D 웹 체험 서비스'**를 완성해낸 이 실행력으로 **A+ 성적**을 거두었으며, 이러한 도전정신은 급변하는 기술 시장에서 드림라인의 신규 서비스를 기획하는 원동력이 될 것입니다.

**[Technical Architect: 보안과 구조를 아는 기획]**
화려함 뒤에는 단단함이 있어야 합니다. 저는 **MediaPolytech**의 백엔드 리드로서 **Spring Boot** 환경에서 **SOLID 원칙**을 적용해 코드를 리팩토링하고, **Spring Security와 JWT** 기반의 인증 시스템을 직접 구축했습니다. 이러한 아키텍처 설계 경험은 드림라인의 B2B/IDC 서비스 기획 시, 보안 리스크를 사전에 식별하고 유지보수가 용이한 구조를 제안하는 데 실질적인 도움이 될 것입니다.

## 3. 입사 후 포부: 데이터 무결성을 지키는 스마트한 기획자가 되겠습니다.
**[Data Integrity: 동아리에서 깨닫고 실무로 완성하다]**
입사 후 저는 **'데이터 무결성(Data Integrity)'에 집착하는 기획자**가 되겠습니다. 2022년 전역 직후 참여한 **연세대 AI/DATA 동아리** 활동을 통해 "Garbage In, Garbage Out"이라는 데이터의 본질을 깨달았고, 이후 **(주)코드비전**에서 데이터 라벨러로 근무하며 이 가치를 실무 품질 검수(QA) 역량으로 발전시켰습니다. 드림라인의 서비스는 신뢰성이 생명인 만큼, 작은 데이터 오류나 프로세스 결함도 용납하지 않는 꼼꼼함으로 최상의 서비스 품질을 기획하겠습니다.

**[Enterprise AI 활용을 통한 효율 실천]**
또한, 제가 검증한 Vibe Coding 경험을 드림라인의 보안 정책과 클라우드 환경에 맞춰 유연하게 적용하겠습니다. Azure OpenAI나 Google Vertex AI와 같이 **기업 보안이 보장된 Enterprise AI 환경** 내에서 기획 업무의 효율을 높이는 방안을 모색하고 실천하겠습니다. 최신 AI 기술을 맹목적으로 도입하는 것이 아니라, 드림라인의 인프라 환경에 최적화된 방식으로 활용하여 팀의 생산성에 기여하고 글로벌 파트너사와의 소통 속도를 높이는 '스마트한 팀원'이 되겠습니다.
