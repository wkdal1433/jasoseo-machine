# <img src="https://unpkg.com/lucide-static@latest/icons/library.svg" width="32" height="32" valign="bottom" /> 자소서 머신 사용 설명서 (Part 3: 나의 무기 만들기)

환경 설정이 끝났다면 AI가 나를 대신해 글을 쓸 수 있도록, **경험 데이터를 입력**할 차례입니다.

---

## <img src="https://unpkg.com/lucide-static@latest/icons/box.svg" width="24" height="24" valign="bottom" /> 3-1. 매직 온보딩 (Magic Onboarding)

이력서나 자기소개서 파일을 업로드하면 AI가 이를 분석해 핵심 프로필과 에피소드를 자동 추출합니다.

1. 대시보드 → **[✨ 매직 온보딩]** 버튼 클릭
2. PDF 또는 DOCX 파일 드래그 드롭
3. AI가 파일을 분석하는 동안 대기 (30~60초)
4. 이름, 학력, 기술 스택, 프로젝트 경험이 자동으로 정리됩니다

## <img src="https://unpkg.com/lucide-static@latest/icons/wand-sparkles.svg" width="24" height="24" valign="bottom" /> 3-2. 에피소드 발굴 마법사 (AI 인터뷰)

추출된 내용이 부족하거나 새로운 경험을 추가하고 싶을 때 사용합니다.

1. **내 라이브러리** → **에피소드** 탭 → **[✨ 새 에피소드 발굴하기]** 클릭
2. AI가 질문을 던집니다: *"최근에 기술적으로 가장 어려웠던 경험은 무엇인가요?"*
3. 자연스럽게 대화하듯 답변 — AI가 꼬리 질문으로 **S-P-A-A-R-L 구조**로 정리해 줍니다
4. 인터뷰 완료 시 구조화된 에피소드 카드가 라이브러리에 저장됩니다

> **S-P-A-A-R-L**: Situation(상황) → Problem(문제) → Action(행동) → Achievement(성과) → Reflection(반성) → Learning(교훈)

## <img src="https://unpkg.com/lucide-static@latest/icons/archive.svg" width="24" height="24" valign="bottom" /> 3-3. 에피소드 라이브러리 관리

저장된 에피소드들은 자소서 생성 시 AI가 골라 쓰는 **원천 데이터**가 됩니다.

- **에피소드 카드**: 제목, 직무 역량, HR 의도(실행력/성장성/안정성/협업) 태그 포함
- **상세 보기**: 카드 클릭 → AI 수정 요청 가능
- **보강 인터뷰**: 미완성 에피소드 → [인터뷰로 완성하기] 버튼
