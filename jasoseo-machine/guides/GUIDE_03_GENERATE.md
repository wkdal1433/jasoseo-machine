# <img src="https://unpkg.com/lucide-static@latest/icons/wand-sparkles.svg" width="32" height="32" valign="bottom" /> 자소서 머신 사용 설명서 (Part 4 & 5: 실전 작성 및 자동 제출)

에피소드 라이브러리가 준비되었다면, **9단계 AI 파이프라인**으로 자소서를 완성하고 채용 사이트에 자동 제출합니다.

---

## <img src="https://unpkg.com/lucide-static@latest/icons/wand-sparkles.svg" width="24" height="24" valign="bottom" /> Part 4. 자소서 작성 마법사

### 4-1. 새 지원서 시작

사이드바 **[새 지원서]** 버튼 클릭 → 두 가지 모드 선택:

| 모드 | 설명 | 권장 상황 |
|------|------|----------|
| **스마트 URL 모드** | 채용 사이트 URL 입력 → 공고·문항 자동 수집 | 공개된 채용 공고 |
| **직접 입력 모드** | 채용공고 텍스트 직접 붙여넣기 | 비공개 공고 / 오프라인 |

### 4-2. 9단계 AI 파이프라인

분석 시작 후 AI가 단계별로 자동 실행됩니다:

| 단계 | 내용 |
|------|------|
| **Step 0** | 기업 리서치 — HR 의도 분석 + 작성 전략 수립 |
| **Step 1** | 질문 재해석 — 기업 가치에 유리하게 재정의 |
| **Step 2** | 에피소드 매칭 — 라이브러리에서 최적 경험 추천 → **[승인]** 필요 |
| **Step 3~5** | 실시간 스트리밍 생성 — 두괄식 도입 → S-P-A-A-R-L 본문 → 마무리 |
| **Step 6~8** | 3중 검증 — 할루시네이션 방지 / 탈락 패턴 제거 / 이중 코딩 검증 |

> **이중 코딩**: 모든 핵심 문장이 *내 이야기(Code 1)* + *기업 가치(Code 2)*를 동시에 전달하는지 검증

### 4-3. 인라인 편집 (Surgical Edit)

생성 완료 후 특정 문장만 선택 → AI에게 부분 수정 요청 가능합니다.

## <img src="https://unpkg.com/lucide-static@latest/icons/rocket.svg" width="24" height="24" valign="bottom" /> Part 5. 채용 사이트 자동 입력

### 5-1. 전체 검토 (Full Review)

모든 문항 완성 후 **Full Review** 화면에서 전체 자소서를 최종 확인하고 편집합니다.

### 5-2. Chrome 확장으로 자동 입력

1. Chrome에서 채용 사이트 **자소서 입력 페이지**로 이동
2. 자소서 머신 앱에서 **[확장으로 전송]** 버튼 클릭
3. Chrome 확장이 폼을 자동 감지 → 각 문항 자동 입력

> 확장 프로그램이 설치되지 않은 경우 [설치 가이드](./GUIDE_01_SETUP.md#2-2-chrome-확장-프로그램-등록-채용-사이트-자동-입력용)를 참조하세요.
