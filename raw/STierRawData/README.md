# STierRawData

이 폴더는 **S급 합격 자기소개서 분석의 근거 데이터**를 저장한다.

---

## 사용 규칙

| 규칙 | 설명 |
|------|------|
| **직접 인용 금지** | 자소서 작성 시 이 파일의 문장을 그대로 사용하지 않는다 |
| **규칙 생성의 출처로만 사용** | 패턴 검증, 규칙 도출의 근거로만 활용 |
| **일반화는 analysis 단계 필수** | 모든 패턴 추출은 `analysis/` 폴더를 거쳐야 한다 |

---

## 데이터 흐름

```
[이 폴더: raw/STierRawData/]
        ↓
    패턴 분석
        ↓
[analysis/company_story_fit.md]
        ↓
    규칙 도출
        ↓
[appendix/APPENDIX_C_COMPANY_VALUES.md]
        ↓
    자소서 작성에 적용
```

## 경고

> **이 폴더의 데이터는 "증거"이지 "권위"가 아니다.**
>
> - 새로운 기업 분석 시: 이 폴더에 추가

> - 패턴 재해석 시: `analysis/`를 수정
