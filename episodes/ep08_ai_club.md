## STAR 구조

### S (상황)
- AI 모델을 '사용'하는 수준이 아니라, 구조와 원리를 이해하고자 함
- 단순 API 호출이 아닌 **모델 내부 동작** 이해 필요성 인식

### T (과제)
- 최신 ML/DL 모델의 작동 원리를 **이론 + 실습**으로 내재화
- 논문 기반 모델 구조 분석 역량 확보

### A (행동)
- **논문 기반 모델 구조 분석**: ResNet, VGG, Diffusion 등
- **직접 모델 구현 및 실험** 수행
- 스터디 내 발표 및 토론 참여

### R (결과)
- 모델 구조에 대한 **해석력 향상**
- AI 알고리즘에 대한 **기술적 대화 가능 수준** 확보
- GIGO 원칙 등 데이터 품질 관점 습득 → Episode 03과 연결

## 자소서 작성 시 주의사항

### DO (이렇게 써라)
- "모델 구조를 **이론적으로 이해**하고 있음"
- "R&D / 기술 기반 직무에 **빠르게 적응 가능**한 학습형 인재"
- "논문 기반 학습 → 직접 구현까지 **End-to-End 학습 경험**"

### DON'T (이렇게 쓰지 마라)
- "동아리에서 AI 공부했다" (무엇을, 어떻게 공부했는지가 중요)
- "CNN, ResNet 알아요" (아는 것과 이해하는 것은 다름)
