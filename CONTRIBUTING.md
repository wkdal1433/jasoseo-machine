# <img src="https://raw.githubusercontent.com/lucide-icons/lucide/main/icons/users.svg" width="32" height="32" valign="bottom" /> Contributing to Jasoseo Machine

먼저 **자소서 머신** 프로젝트에 관심을 가져주셔서 감사합니다! 이 프로젝트는 AI 기술을 통해 취업 준비생들의 고통을 줄이기 위해 시작되었습니다. 여러분의 기여는 이 도구를 더욱 강력하고 신뢰할 수 있게 만듭니다.

기여를 시작하기 전에 아래의 가이드라인을 확인해 주세요.

---

## <img src="https://raw.githubusercontent.com/lucide-icons/lucide/main/icons/book-open.svg" width="24" height="24" valign="bottom" /> Code of Conduct
우리는 서로 존중하며 협력하는 커뮤니티를 지향합니다. 모든 소통은 정중하게 이루어져야 하며, 비하 발언이나 공격적인 태도는 허용되지 않습니다.

## <img src="https://raw.githubusercontent.com/lucide-icons/lucide/main/icons/bug.svg" width="24" height="24" valign="bottom" /> Bug Reports
버그를 발견하셨나요? [Issues](https://github.com/your-repo/issues) 탭에 다음 내용을 포함하여 보고해 주세요.
- **버그 요약:** 어떤 문제가 발생하나요?
- **재현 방법:** 문제를 재현하기 위한 단계를 순서대로 적어주세요.
- **예상 결과 vs 실제 결과:** 무엇을 기대했고, 어떤 결과가 나왔나요?
- **환경 정보:** OS 버전, Node.js 버전, 사용한 AI 모델 (Gemini/Claude).

## <img src="https://raw.githubusercontent.com/lucide-icons/lucide/main/icons/sparkles.svg" width="24" height="24" valign="bottom" /> Feature Requests
새로운 기능 아이디어가 있다면 언제든지 제안해 주세요! 제안 시 해당 기능이 왜 필요한지, 어떤 가치를 주는지 구체적으로 설명해 주시면 논의에 큰 도움이 됩니다.

## <img src="https://raw.githubusercontent.com/lucide-icons/lucide/main/icons/git-pull-request.svg" width="24" height="24" valign="bottom" /> Pull Request (PR) Process
코드를 직접 수정하여 기여하고 싶으시다면 아래 절차를 따라주세요.

1.  이 레포지토리를 **Fork** 합니다.
2.  새로운 기능이나 버그 수정을 위한 **Branch**를 생성합니다. (`feat/new-feature` 또는 `fix/bug-id`)
3.  코드를 수정합니다. 이때 기존의 **코드 스타일**과 **프로토콜**(예: `ANTI_HALLUCINATION.md`)을 엄격히 준수해야 합니다.
4.  가능하다면 수정 사항에 대한 **테스트**를 수행합니다.
5.  수정이 완료되면 원본 레포지토리의 `main` 브랜치로 **Pull Request**를 보냅니다.
6.  관리자의 리뷰를 거친 후 프로젝트에 반영(Merge)됩니다.
