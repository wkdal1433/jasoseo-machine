# 🧙‍♂️ 자소서 머신 v21.1: "The Hands of God"

> **"프로그램이 AI의 닻(Anchor)이 되고, AI가 당신의 신의 손(Hand)이 되는 곳."**  
> 단순한 생성형 AI 챗봇을 넘어, **에이전틱 워크플로우(Agentic Workflow)**와 **인지 아키텍처(Cognitive Architecture)**가 결합된 대한민국 최고 사양의 지능형 채용 자동화 플랫폼입니다.

---

## 💎 아키텍처의 정수 (High-End Engineering)

우리 앱은 업계에서 가장 앞선 기술적 철학을 담고 있습니다.

### 1. 🧠 에이전틱 워크플로우 (Agentic Workflow)
단순한 Q&A가 아닙니다. 프로그램이 AI를 정교한 도구처럼 사용하여 **전략 수립 -> 기업 사냥 -> 데이터 정독 -> 초안 생성 -> 정밀 검증**으로 이어지는 9단계 파이프라인을 자율적으로 수행합니다. AI가 '생각'하고 프로그램이 '통제'하는 완벽한 협업 아키텍처입니다.

### 2. ⚓ 인지 아키텍처 & 컨텍스트 앵커링 (Cognitive Architecture)
AI의 고질적인 문제인 '비정합성'을 **컨텍스트 닻(Context Anchor)** 시스템으로 해결했습니다. AI 스스로 본인의 분석 상태를 기록하고 복구하는 '자의식 기반 세션 관리'를 통해, 그 어떤 돌발 상황에서도 대화의 맥락이 단 1%도 손실되지 않습니다.

### 3. 🛡️ 하이브리드 보안 브릿지 (Hybrid Secure Bridge)
데스크탑 앱의 강력한 성능과 브라우저의 접근성을 **로컬 Express 서버**로 연결했습니다. **HMAC SHA-256 서명**과 **타임스탬프 검증**을 거친 3중 보안망을 통해, 오직 사용자만이 허용한 데이터만이 브라우저로 안전하게 흐릅니다.

### 4. 🛰️ 실시간 인스펙터 콘솔 (Real-time Inspector)
AI의 뇌 속을 들여다보는 것 같은 투명함을 제공합니다. 백엔드에서 일어나는 모든 CLI 로그와 도구 호출 현황을 **매트릭스 스타일의 터미널 UI**로 실시간 스트리밍하여, 작업의 신뢰도를 극한으로 끌어올렸습니다.

## 🌟 핵심 기능 (The Core Modules)

*   **AI-Native 온보딩**: 라이브러리 의존성 제로. AI가 직접 파일 시스템을 읽어 PDF/MD에서 12개 섹션 프로필을 정밀 추출합니다.
*   **에피소드 발굴 마법사**: 사용자님의 인생 데이터에서 보석 같은 스토리를 무한대로 발굴하고, 인터뷰를 통해 S-P-A-A-R-L 구조로 정제합니다.
*   **1초 일괄 주입**: Chrome Extension 연동을 통해 복사-붙여넣기 없이 모든 입력칸을 1초 만에 촥! 채워주는 '신의 손' 기능을 제공합니다.
*   **엔진 핫스왑 (Hot-Swap)**: 토큰 소진이나 쿼터 제한 발생 시, 1초 만에 Claude에서 Gemini로(혹은 그 반대로) 엔진을 갈아타며 중단 없는 작업을 보장합니다.

## 🛠 기술 스택 (The Tech Stack)

*   **Core**: Electron, electron-vite, Node.js (Express Bridge)
*   **Frontend**: React 19, TypeScript, Tailwind CSS, Zustand, Framer Motion
*   **AI Brain**: Gemini 1.5/2.0 Pro, Claude 3.5 Sonnet (Agentic CLI Integration)
*   **Hand**: Chrome Extension (Manifest v3, SubtleCrypto HMAC Auth)
*   **Architecture**: State-Aware Context Injection Pipeline, Multi-Agent Orchestration

## 📖 사용 가이드 (Workflow)

1.  **[온보딩]**: 기존 자소서 PDF를 드래그하세요. AI가 12개 섹션 프로필을 즉시 구성합니다.
2.  **[에피소드]**: 발굴 마법사를 통해 본인의 경험을 S-P-A-A-R-L 자산으로 변환합니다. (자동 세션 복구 지원!)
3.  **[작성]**: 지원할 기업을 정하고 AI와 함께 9단계 전략 작성을 진행합니다.
4.  **[주입]**: 확장 프로그램을 켜고 '자동 입력' 버튼을 누르세요. 1초 만에 제출 준비가 끝납니다.
