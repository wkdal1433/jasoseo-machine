# 🚀 AI 자소서 자동화 머신 (Jasoseo Machine v5.0)

> **"공채 시즌의 고통을 0으로."**  
> 전략 수립부터 실시간 기업 분석, 그리고 단 1초 만의 일괄 입력까지.  
> 2026년 상반기 취업 시장을 선도하는 지능형 에이전트 시스템입니다.

---

## 🌟 핵심 기능 (The Triple-Core)

### 1. 📂 12개 섹션 통합 프로필 시스템
*   단순 인적사항을 넘어 병역, 어학, 인턴, 수상 경력, 기술 스택 등 **채용 사이트의 모든 항목**을 완벽히 커버합니다.
*   한 번의 입력으로 수십 개의 기업에 동일한 데이터를 재사용하는 **'데이터 자산화'**를 실현합니다.

### 2. 🔍 지능형 기업 분석 에이전트 (2026-Anchored)
*   **실시간 시점 동기화**: 오늘 날짜(예: 2026-03-09)를 기준으로 최신 공고를 인터넷에서 사냥합니다.
*   **JD 딥러닝 분석**: 모집요강과 우대사항을 분석하여 사용자 에피소드와의 **매칭 점수**를 계산하고 전략을 제안합니다.
*   **유저 컨펌 워크플로우**: AI가 찾은 정보를 유저가 직접 검토하고 선택하는 '인간-AI 협업' 구조입니다.

### 3. ⚡ 일괄 주입 프록시 에이전트 (Batch Input Proxy)
*   **1초의 기적**: 50~100개의 입력 필드를 단 하나의 자바스크립트 스니펫으로 **한 번에 일괄 주입**합니다.
*   **프레임워크 대응**: React, Vue 등 최신 웹 프레임워크의 상태 업데이트(State Sync)를 완벽히 지원합니다.
*   **보안 우회**: 브라우저 콘솔을 활용하여 강력한 채용 사이트 보안 모듈을 안전하게 통과합니다.

## 🛠 기술 스택 (Tech Stack)

*   **Frontend**: React 19, TypeScript, Tailwind CSS, Zustand (State Management)
*   **Desktop**: Electron, electron-vite
*   **AI Engine**: Gemini 2.0 Pro / Claude 3.5 Sonnet (via CLI Subprocess)
*   **Automation**: Intelligent Batch Script Generator & Event Simulator
*   **Database**: JSON-based Lightweight Database (Local-First)

## 📖 사용 가이드 (Workflow)

1.  **[설정]**: 12개 섹션의 **프로필**을 작성하고, 본인의 **경험 에피소드**를 등록합니다.
2.  **[분석]**: 지원할 **기업명**을 입력합니다. AI가 2026년 최신 공고와 인재상을 가져오면 확인(Confirm)합니다.
3.  **[작성]**: AI가 추천하는 에피소드와 전략을 바탕으로 **자기소개서 초안**을 생성하고 검토합니다.
4.  **[입력]**: 
    *   채용 사이트의 `<body>` 내용을 복사하여 앱에 붙여넣습니다.
    *   생성된 **'자동 입력 스크립트'**를 복사합니다.
    *   브라우저 개발자 도구(F12) 콘솔에 붙여넣고 **Enter**를 누릅니다.
5.  **[완료]**: 촥촥촥 채워진 데이터를 확인하고 **제출하기**를 누릅니다!
